### How to run api locally

- Step 1 - Clone the git repository

- Step 2 - Paste .env file into the repository folder

- Step 3 - Open CMD from repository folder and install npm package
  ```
  npm install
  npm run build
  npm start
  ```
- Step 4 - Open link in the browser
  ```
  http://localhost:5000/
  ```
